1. Soma até estabilizar

Peça números ao usuário repetidamente até que ele digite dois números iguais seguidos.
Retorne a soma de todos os números digitados.

2. Número primo “reforçado”

Leia um número e determine se ele é primo sem usar for, apenas com while.
Você deve testar todos os divisores possíveis manualmente.

3. Sequência de Collatz

Dado um número n, gere toda a sequência de Collatz usando apenas while:

Se for par: n = n / 2

Se for ímpar: n = 3n + 1

Pare quando chegar em 1 e conte quantos passos levou.

4. Maior sequência crescente

O usuário entra vários números (termina com 0).
Descubra qual foi o maior trecho consecutivo de números crescentes.

Exemplo:
Entrada → 2 4 1 2 3 9 0
Resposta → trecho mais longo = 1 2 3 9 (4 números)

5. Adivinhe o número com limite de tentativas

O programa escolhe um número aleatório de 1 a 100.
O usuário tem apenas 7 tentativas para acertar.
Dê dicas: "maior" ou "menor".

6. Detectar palíndromo sem strings

Leia um número tipo 12321 e determine se é palíndromo sem converter para string.
Use while para inverter o número matematicamente.

7. Fatorial de grande porte

Calcule o fatorial de um número grande (ex: 50), usando while
e sem usar nenhuma função pronta.

8. Jogo do maior de 3

O usuário deve inserir números até digitar três números decrescentes consecutivos.
Enquanto isso, o programa deve manter atualizado o maior número já digitado.

9. Verificar se um número é perfeito

Um número perfeito é a soma de seus divisores (ex: 6 = 1+2+3).
Faça isso sem usar for, apenas com while.

10. Sequência customizada

O usuário escolhe:

número inicial

número final

operação (somar, subtrair, multiplicar ou dividir)

passo

O programa deve gerar e mostrar a sequência usando while
até alcançar ou ultrapassar o limite.
