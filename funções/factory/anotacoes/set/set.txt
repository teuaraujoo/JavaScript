- liga a propriedade da função para ser chamada quando existe uma tentativa de definir valor para essa propriedade.
- pode ser usado para executar uma função sempre que se tenta mudar uma propriedade específica.
- set -> definir

SINTAXE:
set prop(val) { . . . }

prop -> nome da propriedade ligada a função.

val -> um apelido para variável que será atribuidada a prop.

var expr = "foo";

var obj = {
  baz: "bar",
  set [expr](v) {
    this.baz = v;
  },
};

console.log(obj.baz); // "bar"
obj.foo = "baz"; // run the setter
console.log(obj.baz); // "baz"
