⭐ 10. DESAFIO — map + filter + reduce + arrow function
Array base:
const produtos = [
  { nome: "Mouse", preco: 80 },
  { nome: "Teclado", preco: 150 },
  { nome: "Monitor", preco: 900 },
  { nome: "Headset", preco: 120 }
];

Solução completa:
const calcularTotal = produtos => 
  produtos
    // 1. Aplica 15% de desconto
    .map(p => ({
      nome: p.nome,
      preco: p.preco * 0.85
    }))
    // 2. Filtra apenas os acima de R$100
    .filter(p => p.preco > 100)
    // 3. Soma os preços
    .reduce((total, p) => total + p.preco, 0)
    // 4. Arredonda
    .toFixed(2);


✔ Arrow function completa
✔ Retorno encadeado
✔ Desafio atendido

⭐ 10. DESAFIO HARD — Agrupar vendas e ranquear vendedores
Solução completa e comentada
const analisarVendas = vendas => {
  
  // 1 e 2. Agrupar por vendedor e somar total
  const totais = vendas.reduce((acc, venda) => {
    acc[venda.vendedor] = (acc[venda.vendedor] || 0) + venda.valor;
    return acc;
  }, {});

  // 3. Converter em array de objetos
  const resultado = Object.entries(totais).map(([vendedor, total]) => ({
    vendedor,
    total
  }));

  // 4. Ordenar do maior para o menor
  resultado.sort((a, b) => b.total - a.total);

  // 5. Melhor vendedor
  const melhor = resultado[0];

  return { resultado, melhor };
};
